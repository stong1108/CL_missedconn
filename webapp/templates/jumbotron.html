Missed Connections Explorer

Missed Connections is a section of Craigslist where people write about strangers they notice, usually to reconnect.
This web app is a tool for exploring the Missed Connections collected for an NLP project.
To find out more about Content Analysis of Missed Connections, please visit the github repo.
